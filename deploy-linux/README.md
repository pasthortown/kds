# Deploy Linux (Docker)

Para desplegar en Linux con Docker, consulta la guia completa en:
- [docs/DEPLOY_LINUX.md](../docs/DEPLOY_LINUX.md)

## Despliegue Rapido

```bash
# Desde la raiz del proyecto
docker-compose up -d

# O usando este archivo
docker-compose -f deploy-linux/docker-compose.yml up -d
```
